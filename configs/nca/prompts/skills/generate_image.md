# Generate_Image Tool

Use Generate_Image to create images from text descriptions.

## When to Use
- User asks to create, generate, or make an image
- User wants a visual representation of something
- User needs an illustration or artwork

## Parameters
- Defaults to "nano banana" if no prompt provided
- Provide clear, descriptive prompts for best results

## Best Practices
- Use detailed descriptions for better image quality
- Specify style, mood, or artistic direction when relevant
- Explain what image you're generating
